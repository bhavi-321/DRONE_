# Autonomous UAV Systems Portfolio

A collection of projects focused on **autonomous drones, computer vision, and real-world UAV systems**.  
This repository showcases work done across competitions and independent development.

---

##  Overview

I work on building **real-time intelligent systems for drones**, combining:
- Computer Vision  
- Autonomous Control  
- UAV Systems Integration  

This repo includes projects developed as part of:
- SAE Autonomous Drone Development Challenge (ADDC)
- IROC

## Projects

**Focus:** UAV control systems, telemetry GUI, and real-time vision integration

- Developed **real-time telemetry GUI** for monitoring flight parameters

**Achievement:**
